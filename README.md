To-Do List
A simple to-do list application built with Python.

Requirements
Python 3.6 or later
pytest
Installation
Clone the repository and install the dependencies:

git clone the repo

cd to-do-list
pip install -r requirements.txt
Usage
To run the to-do list application:

Copy code
python todo.py
To run the tests:

Copy code
pytest
